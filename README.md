# 당뇨수첩
 일일 혈당량과 식사, 투약 기록을 입력하고 관리하세요.
- URL: https://diabetes-note.netlify.app/

## ✏️기능

1. 혈당수치를 기록할 수 있습니다. 
2. 목표범위를 벗어난 혈당 수치는 색상을 통하여 강조표시됩니다.
3. 투약기록을 기록할 수 있습니다.
4. 공공API를 이용하여 영양성분을 조회하고 식단을 기록할 수 있습니다.
